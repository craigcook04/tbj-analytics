About Me

I chose to complete task one for my Blue Jays analytics application since the requirements are most similar to the application outlined in my resume, and I wanted to showcase the style and functionality of my web design.

My original application was built using Angular, so my experience developing this application for Django was very enjoyable, especially because of the API resources that were provided. I am extremely interested in viewing real MLB data, and creatively displaying that information in a modern web page view was a fun challenge.

Now that I have access to these resources, I am enthusiastic about scaling my project to include all three tasks in my web application.

I would appreciate any feedback you may have for my current application, and would be very excited for the opportunity to develop any new feature requests.

Thank you for your time in viewing my application.
